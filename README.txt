The Sports Club Management System is a web-based application designed to simplify and digitalize the management activities of sports clubs.
This system allows administrators to manage members, fees, attendance, and club activities efficiently through an interactive and user-friendly interface.

- Key Features

* Member Management
  Add, edit, view, and manage club members with full profile details.
* Attendance Tracking
  Record and monitor member attendance for training sessions and events.
* Payment & Fees Management
  Track membership fees, due payments, and payment status.
* Interactive Dashboard
  View overall statistics such as total members, fees received, and upcoming events.
* Admin Login System
  Secure authentication for administrators.
* User-Friendly Interface
  Built with responsive and clean UI for smooth navigation.

- Tech Stack Used
Frontend: HTML, CSS, JavaScript, Bootstrap
Backend: PHP
Database: MySQL
Tools: VS Code, phpMyAdmin
